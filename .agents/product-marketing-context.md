# Product Marketing Context — Wrangle

## Product
- **Name:** Wrangle
- **URL:** https://wrangleapp.dev
- **One-liner:** The markdown editor built for AI-native development
- **Category:** Developer tools / macOS native app

## What It Does
Native macOS markdown editor purpose-built for developers working with AI agents (Claude Code, Gemini, etc.). Combines a Typora-style inline-rendered markdown editor with embedded terminals, smart notifications, and first-class understanding of AI config files (CLAUDE.md, SKILL.md, AGENTS.md, system prompts).

## Key Features
1. **Embedded terminals** — Run AI agent sessions in tabbed terminals without leaving the app
2. **Smart notifications** — Get pulled back to the right terminal when an agent needs input or finishes
3. **Session context** — See linked Claude file, active skills, and MCP servers per session
4. **Token counting** — Always visible in status bar for prompt files
5. **XML-in-markdown** — Highlights/collapses `<tools>`, `<instructions>`, `<system>` blocks
6. **AI file awareness** — Distinct treatment for CLAUDE.md, SKILL.md, AGENTS.md files
7. **Native macOS** — Swift, SwiftUI, Apple Silicon. Not Electron.
8. **Fuzzy finder** (Cmd+P) — Quick file access across all bookmarked projects
9. **File-first** — No import/export, no proprietary formats. Just .md files on disk.

## Target Audience
- **Primary:** Developers who work with AI agents daily (Claude Code, Gemini CLI, Cursor, etc.)
- **Niche size:** ~10,000 highly technical people who edit CLAUDE.md files and system prompts regularly
- **Platform:** macOS only (Apple Silicon, macOS 15+ Sequoia)
- **Pain point:** VS Code is not a markdown editor — XML breaks, no token counting, raw markdown rendering, no agent notifications

## Competitors / Alternatives
- VS Code (with markdown preview) — code editor with markdown bolted on
- Obsidian — personal knowledge base, not developer-focused
- Typora — great markdown editor but no AI awareness or terminals
- iA Writer — writing-focused, no developer features
- Cursor — AI code editor, not markdown-focused
