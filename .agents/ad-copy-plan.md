## 4. Success Metrics

**Google Ads:**
- **CTR:** Target >3% (niche keywords should perform well)
- **CPC:** Track average, aim to stay under $2
- **Conversions:** Set up conversion tracking on the /buy page (or LemonSqueezy thank-you redirect)
- **ROAS:** At $19/sale, you need ~4 sales/mo to break even on Google's $60. That's a ~1% conversion rate on ~400 clicks.

**X Ads:**
- **Engagement rate:** Target >1.5% (likes + replies + link clicks / impressions)
- **Link clicks:** Track click-through to wrangleapp.dev
- **Cost per link click:** Aim for <$1
- **Profile follows:** Secondary metric â€” builds long-term audience

**Overall:**
- **CAC (customer acquisition cost):** Track total spend / total new purchases. Target <$12 (allows profitable acquisition at $19 price point).
- **Attribution:** Use UTM parameters on all ad links (`?utm_source=google&utm_medium=cpc&utm_campaign=markdown-editor` etc.) to separate channels in analytics.

## 5. Optimization Cadence

### Week 1-2 (Launch)
- Launch all 3 Google ad groups and 2-3 X tweets simultaneously
- Set up conversion tracking (LemonSqueezy webhook or thank-you page pixel)
- Check daily for disapprovals or billing issues

### End of Month 1
- **Google:** Pause any ad group with CTR <1.5%. Double down on the best-performing one. Test new headline variations for winners.
- **X:** Kill bottom 2 tweets. Promote the top performer with remaining budget. Note which style (pain point vs screenshot vs comparison) resonated.
- **Decision:** If Google ROAS >1, shift budget to 70/30 Google/X. If X engagement is strong but Google isn't converting, flip to 40/60.

### Month 2
- Add new ad variations based on Month 1 learnings
- Test one new X tweet style (e.g., short video/GIF of the notification flow)
- Start building a keyword negative list from search term reports (Google shows you what people actually typed)

### Month 3
- Evaluate whether $100/mo spend is worth continuing at current conversion rate
- If CAC >$12, pause paid ads and reallocate to content/SEO or direct outreach
- If CAC <$6, consider increasing budget to $150-200/mo

### Ongoing
- Monthly: review search terms, update negatives, refresh ad copy
- Quarterly: reassess budget split and total spend
- Always: screenshot top-performing ads before changing them (you'll want to reference what worked)
